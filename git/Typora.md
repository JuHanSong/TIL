# Typora 

## 1. 마크다운(MarkDown) 문법이란?

+ 문서를 쉽고 빠르게 체계적으로 정리할 수 있도록 고안된 문법. 확장자가 .md 로 된파일들이 마크다운 문법으로 작성된 파일.

## 2. 왜 사용해야하는지?

* 텍스트에 서식을 지정할 수 있어서 문서를 보기좋게 정리할 수 있음.
* 코드블럭을 지원하기 때문에 깃허브 리드미 파일, 노션 등 많은 개발 플랙폼에서 사용.
* html로 변환이 가능.

## 3. 마크다운 에디터?

* 문법을 외우지 않아도 단축키로 서식 지정가능.
* 메모장보다 기능이 많고 워드보다 가볍고 빠름.
* .pdf, .word, .html, .txt 파일로 저장 및 내보내기 가능.

## 4. typora 설치하기

* 공식 사이트 [https://www.typora.io/] 에서 다운로드 가능.

## 5. 마크다운 문법 및 타이포라 단축키

- 제목(h1 ~ h6) : Ctrl + 1, 2, 3, 4

- 순서가 없는 목록 : Ctrl + Shift + [

- 순서가 있는 목록 : Ctrl + Shift + ]

- 코드블록 : Ctrl + Shift + K

  ```
  printf("hello world"); // 코드블럭 테스트
  ```

* 코드 : ''  구문을 감싸기

* 인용문 : Ctrl + Shift + Q

* 링크 : 웹주소 복사한 뒤 링크 넣을 텍스트 드래그 한 후 Ctrl + K

* 그 외 :

  ```
  Ctrl + X,C,V,Z,S / Ctrl + b / Ctrl + i 
  ```

  같이 워드에서 쓰던 단축키들도 전부 사용 가능. 

  아래 링크를 참고하여 공식 문서를 참고.
